# Роль: Администратор

## ❓ Что это за роль?

**Ответ:** Администратор (`admin`) — это роль с максимальными правами в модуле "Пользователи". Администратор имеет полный доступ к управлению всеми пользователями, ролями, командами и отделами компании.

## ❓ Кому назначается роль администратора?

**Ответ:** Роль администратора находится только у разработчиков, которым необходим полный контроль над управлением пользователями и организационной структурой.

## ❓ Что может делать администратор с пользователями?

**Ответ:** Администратор имеет полные права на все операции с пользователями:

### Управление пользователями
- Просмотр всех пользователей системы
- Редактирование любых пользователей
- Создание новых пользователей
- Удаление пользователей
- Назначение любых ролей, включая роль администратора
- Удаление пользователей из команд
- Управление отделами, командами, должностями и категориями
- Просмотр и редактирование ставок всех сотрудников


## ❓ Может ли администратор видеть ставки всех сотрудников?

**Ответ:** Да, администратор имеет разрешение, которое позволяет просматривать почасовые ставки всех сотрудников компании в панели управления пользователями.

## ❓ Может ли администратор удалять пользователей?

**Ответ:** Да, администратор может удалять пользователей полностью из системы.

## ❓ Может ли администратор удалять пользователей из команд?

**Ответ:** Да, администратор может удалять пользователей из команд, не удаляя их из системы полностью. Это позволяет:
- Переводить сотрудников между командами
- Временно исключать из команды
- Реорганизовывать структуру команд

## ❓ Может ли администратор назначать роль администратора другим пользователям?

**Ответ:** Да, администратор имеет специальное разрешение на назначение роли администратора другим пользователям. Это критически важное разрешение, которое позволяет:
- Делегировать административные полномочия
- Назначать новых администраторов системы
- Передавать полный контроль над управлением пользователями

Важно: только администратор может назначить роль администратора. Руководители отделов не имеют такого права.

## ❓ Как администратор управляет командами и отделами?

**Ответ:** Администратор имеет полный набор разрешений для управления организационной структурой:

1. **Отделы** — создание, редактирование, удаление отделов
2. **Команды** — создание и управление командами
3. **Руководители команд** — назначение/смена руководителей команд
4. **Удаление команд** — удаление команд

## ❓ Может ли администратор управлять должностями и категориями?

**Ответ:** Да, администратор имеет следующие разрешения:

создание, редактирование и удаление должностей (например, "Разработчик", "Менеджер")
управление категориями пользователей для их классификации

Это позволяет гибко настраивать организационную структуру компании.

## ❓ Можно ли редактировать ставки сотрудников?

**Ответ:** Да, администратор может не только просматривать ставки всех сотрудников, но и редактировать их через панель управления пользователями.

Это важно для:
- Установки начальных ставок новым сотрудникам
- Корректировки ставок при повышении
- Управления затратами на персонал
