# Роль: Руководитель проекта

## ❓ Что это за роль?

**Ответ:** Руководитель проекта (`project_manager`) — это роль с ограниченными правами в модуле "Пользователи". Основная задача этой роли — управление проектами, а не пользователями.

## ❓ Кому назначается роль руководителя проекта?

**Ответ:** Роль назначается сотрудникам, ответственным за реализацию конкретных проектов.

Важно: назначить роль может администратор.

## ❓ Какие права есть у руководителя проекта в модуле "Пользователи"?

**Ответ:** В модуле "Пользователи" руководитель проекта имеет минимальные права:

### Управление профилем
- Редактирование своего профиля

### Просмотр информации
- Просмотр информации о пользователях
- Просмотр ставок всех сотрудников компании (для планирования бюджета проектов)

## ❓ Может ли руководитель проекта управлять пользователями?

**Ответ:** Нет, руководитель проекта не имеет прав на управление другими пользователями.

## ❓ Может ли руководитель проекта видеть ставки сотрудников?

**Ответ:** Да, руководитель проекта имеет разрешение, которое позволяет видеть почасовые ставки **всех сотрудников компании**.

Зачем это нужно:
- Планирование бюджета проекта
- Оценка стоимости работ по проекту
- Выбор исполнителей с учетом их ставок
- Контроль расходов на проект

Примечание: это единственное разрешение на просмотр данных других пользователей.

## ❓ Есть ли доступ к панели администратора?

**Ответ:** Нет, руководитель проекта не имеет разрешения.

Фокус роли — управление проектами, а не пользователями.

## ❓ Может ли руководитель проекта назначать роли?

**Ответ:** Нет, руководитель проекта не имеет разрешенияи не может назначать или изменять роли других пользователей.

Назначение ролей доступно:
- Администраторам
- Руководителям отделов

## ❓ Какие ограничения есть у руководителя проекта в модуле "Пользователи"?

**Ответ:** Основные ограничения:

- **Не может управлять пользователями** — только свой профиль
- **Не может назначать роли** — это право администрации
- **Не может создавать команды/отделы** — нет административных прав
- **Не может удалять пользователей** — только администратор
- **Нет доступа к панели администратора** — ограниченный функционал

## ❓ Что не может редактировать руководитель проекта в своем профиле?

**Ответ:** Руководитель проекта не может редактировать в своем профиле:

- Роль (может изменить только администрато)
- Ставка (может изменить только руководство)
- Отдел
- Команда

Роль Руководителя проекта фокусируется на управлении проектами, а не людьми.
