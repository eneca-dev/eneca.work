# Роль: Пользователь

## ❓ Что это за роль?

**Ответ:** Пользователь (`user`) — это базовая роль в системе eneca.work с минимальными правами в модуле "Пользователи". Эта роль предназначена для сотрудников, которые используют систему для работы, но не управляют другими пользователями.

## ❓ В чем отличие от других ролей?

**Ответ:** Пользователь — это роль с минимальными правами в модуле "Пользователи":

**Отличия от всех управленческих ролей:**
- Не может управлять другими пользователями
- Не имеет доступа к панели администратора
- Видит только свою ставку
- Не может назначать роли
- Не может редактировать профили других пользователей
- Не может создавать команды или отделы

Основная задача — использование системы для работы.

## ❓ Что может делать пользователь в модуле "Пользователи"?

**Ответ:** Пользователь имеет следующие возможности:

### Управление профилем
- Просмотр и редактирование своего профиля
- Изменение личных данных (ФИО, контакты)
- Просмотр своей ставки

## ❓ Может ли пользователь редактировать профили других?

**Ответ:** Нет, пользователь имеет разрешение, которое позволяет редактировать **только свой профиль**.

## ❓ Может ли пользователь видеть ставки коллег?

**Ответ:** Нет, пользователь имеет разрешение, которое позволяет видеть **только свою ставку**.

Ставки других сотрудников видят:
- Руководители команд (своя команда)
- Руководители отделов (свой отдел)
- Руководители проектов (вся компания)
- Администраторы (вся компания)

## ❓ Есть ли доступ к панели администратора?

**Ответ:** Нет, пользователь не имеет разрешения и не может:
- Заходить в панель администратора
- Управлять пользователями
- Назначать роли

## ❓ Может ли пользователь назначать роли?

**Ответ:** Нет, пользователь не имеет разрешения и не может назначать или изменять роли других пользователей.

Назначение ролей доступно:
- Администраторам
- Руководителям отделов

## ❓ Может ли пользователь удалить себя из системы?

**Ответ:** Нет, пользователь не может удалить свою учетную запись.

Удаление пользователей может выполнять только администратор.

## ❓ Может ли пользователь просматривать информацию о коллегах?

**Ответ:** Базовая информация о коллегах обычно доступна для просмотра (имя, должность, контакты), но:
- Нет доступа к редактированию
- Нет доступа к ставкам
- Нет доступа к полной информации профиля

Детальная информация доступна только через панель администратора, к которой у обычного пользователя нет доступа.

## ❓ Может ли пользователь создавать других пользователей?

**Ответ:** Нет, у пользователя нет разрешения на создание новых пользователей.
