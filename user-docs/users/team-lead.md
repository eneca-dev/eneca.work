# Роль: Руководитель команды

## ❓ Что это за роль?

**Ответ:** Руководитель команды (`team_lead`) — это роль для управления пользователями небольшой рабочей группы (команды). Эта роль предоставляет права на редактирование пользователей своей команды и просмотр их ставок.

## ❓ Кому назначается роль руководителя команды?

**Ответ:** Роль назначается сотрудникам, ответственным за управление рабочей группой.

Важно: назначить роль руководителя команды могут администратор или руководитель отдела.

## ❓ В чем отличие от других ролей?

**Ответ:** Руководитель команды — это роль с ограниченными административными правами:

**Отличия от администратора:**
- Управляет только своей командой
- Не может назначать роли
- Не может удалять пользователей
- Видит ставки только своей команды

**Отличия от руководителя отдела:**
- Работает только с командой, а не с целым отделом
- Не может удалять команды
- Не может назначать руководителей команд

## ❓ Что может делать руководитель команды?

**Ответ:** Руководитель команды имеет следующие возможности в модуле "Пользователи":

### Управление командой
- Просмотр пользователей своей команды
- Редактирование профилей членов команды
- Редактирование собственного профиля
- Просмотр ставок сотрудников команды

### Доступ к панели администратора
- Просмотр списка пользователей (в рамках команды)
- Редактирование данных членов команды

## ❓ Может ли руководитель команды управлять всеми пользователями?

**Ответ:** Нет, руководитель команды имеет доступ только к редактированию пользователей **своей команды**.

## ❓ Может ли руководитель команды назначать роли?

**Ответ:** Нет, руководитель команды не имеет разрешения и не может назначать или изменять роли пользователей.

## ❓ Может ли руководитель команды видеть ставки сотрудников?

**Ответ:** Да, но только своей команды. Руководитель команды имеет разрешение, которое позволяет видеть почасовые ставки сотрудников своей команды.

Зачем это нужно:
- Планирование загрузки команды
- Оценка стоимости работ команды
- Распределение задач с учетом ставок

Важно: руководитель команды НЕ видит ставки сотрудников других команд.

## ❓ Что может редактировать руководитель команды в профилях?

**Ответ:** Руководитель команды может редактировать в профилях членов своей команды только команду соответственно.

## ❓ Есть ли доступ к панели администратора?

**Ответ:** Да, руководитель команды имеет разрешение, которое дает доступ к панели администратора по адресу `/dashboard/users`.

Но в панели руководитель команды видит только:
- Пользователей своей команды
- Ограниченный функционал редактирования

## ❓ Может ли руководитель команды удалять членов команды?

**Ответ:** Нет, руководитель команды не имеет разрешения и не может удалять пользователей из системы.

Удаление пользователей может выполнять только администратор.

## ❓ Может ли руководитель команды создавать новых пользователей?

**Ответ:** Нет, у руководителя команды нет разрешения на создание новых пользователей.

Создание пользователей доступно:
- Администраторам

## ❓ Может ли руководитель команды назначать или снимать себя как руководителя?

**Ответ:** Нет, руководитель команды не может управлять назначением руководителей команд. Это право имеют:
- Администраторы 
- Руководители отделов

## ❓ Какие ограничения есть у руководителя команды?

**Ответ:** Основные ограничения в модуле "Пользователи":

- **Работа только со своей командой** — не видит других пользователей
- **Не может назначать роли** — это право администратора/руководителя отдела
- **Не может удалять пользователей** — только редактирование
- **Не может создавать пользователей** — только работа с существующими
- **Видит ставки только своей команды** — не всей компании

## ❓ Может ли руководитель команды управлять отделами?

**Ответ:** Нет, руководитель команды не имеет разрешения и не может:
- Создавать отделы
- Редактировать отделы
- Удалять отделы

Управление отделами доступно только администраторам.
